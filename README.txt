Викулов Дмитрий М3О-309Б-21

Код является простым калькулятором для выполнения операций сложения, вычитания, умножения и деления над числами, 
предоставляемыми пользователем. 
Код предлагает пользователю ввести знак операции (+, -, *, /) и два числа для выполнения соответствующей операции. 
Программа выполняет операцию и выводит результат.
Используется цикл while True, что означает, что программа будет выполняться до тех пор, 
пока не будет введен знак операции "0", что приведет к завершению программы.
